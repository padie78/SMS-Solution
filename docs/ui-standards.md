# UI/UX & Angular Standards

## State Management
- **Signals First:** Use Angular Signals for UI-bound state and computed values.
- **RxJS Second:** Use RxJS only for complex data streams, WebSockets, or HTTP calls.
- **Store Pattern:** All stores must extend `StateStoreService<T>`.

## Design Tokens (PrimeNG)
- **Primary:** Forest Green (Sustainability).
- **Alerts:** 
    - Critical (Red): High leakage, missing compliance.
    - Warning (Yellow): Anomalous consumption.
    - Success (Green): Saving target achieved.

## Component Architecture (Atomic Design)
- **Atoms:** PrimeNG basic inputs, buttons.
- **Molecules:** Search bars, Metric cards with Signals.
- **Organisms:** Ingestion Wizards, Dashboard Grids.
- **Templates:** Main Layout with Sidebar and NavigationService integration.

## Rule: "The 1-Click Action"
Any "Prescriptive Action" generated by the AI must be actionable via a single button click in the UI.
